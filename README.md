# SC2022

情報処理安全確保支援士試験(2022年)用の学習用サイト。  
Next.jsで作成してみた。  

もともと<https://koko-campus.org>にあったものの一部をNext.jsで置き換えて独立させた。  
※ 元のソースコードは<https://github.com/osawa-koki/koko-campus.net>(private)  

## 実行方法

```shell
# デバグ用実行
yarn dev

# デプロイ用ビルド
yarn build
```

## 開発環境

| モジュール | バージョン |
| ---- | ---- |
| Windows | 11 Home |
| Node.js | 19.3.0 |
